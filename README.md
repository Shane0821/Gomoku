# Gomoku
A gomoku player via MiniMax Search and Alpha-Beta Pruning, running on https://botzone.org.cn/.

## How to use
```bash
mkdir build
cd build
cmake ..
make
./gomoku
```

## Techniques
- MinMAX with Alpha-Beta Pruning.
- Zobrist.
- Transposition Table.
- Iterative deepening.
